    STEP 1 — Update ProjectDetailClientProps and component signature

    The server component that calls ProjectDetailClient already passes `project: ProjectStats`. Since ProjectStats now has slackWebhookUrl, no props change is needed — just use project.slackWebhookUrl.

    STEP 2 — Add state for the Notifications tab

    Inside the ProjectDetailClient function, add two state variables near the existing ones:
    ```typescript
    const [slackWebhookUrl, setSlackWebhookUrl] = useState(project.slackWebhookUrl ?? "");
    const [slackSaving, setSlackSaving] = useState(false);
    ```

    STEP 3 — Add save handler

    ```typescript
    const handleSaveSlackWebhook = async () => {
      setSlackSaving(true);
      try {
        const res = await fetch(`/api/projects/${project.id}`, {
          method: "PATCH",
          headers: { "Content-Type": "application/json" },
          body: JSON.stringify({ slack_webhook_url: slackWebhookUrl || null }),
        });
        if (res.ok) {
          toast.success("Slack webhook saved");
        } else {
          toast.error("Failed to save webhook URL");
        }
      } catch {
        toast.error("Network error");
      } finally {
        setSlackSaving(false);
      }
    };
    ```

    STEP 4 — Add TabsTrigger for Notifications

    After the existing "Alerts" TabsTrigger (value="alerts"), add:
    ```tsx
    <TabsTrigger
      value="notifications"
      className="rounded-lg text-sm data-[state=active]:bg-card data-[state=active]:text-foreground"
    >
      Notifications
    </TabsTrigger>
    ```

    STEP 5 — Add TabsContent for Notifications

    After the closing tag of the "alerts" TabsContent, add a new TabsContent:

    ```tsx
    <TabsContent value="notifications" className="mt-5">
      <div className="border border-border rounded-2xl bg-card p-5 space-y-5">
        <div>
          <h3 className="font-semibold mb-1">Slack Notifications</h3>
          <p className="text-sm text-muted-foreground">
            Paste an incoming webhook URL to receive budget alerts in Slack.
          </p>
        </div>
        <div className="space-y-2">
          <Label htmlFor="slack-webhook" className="text-sm font-medium">
            Slack Webhook URL
          </Label>
          <Input
            id="slack-webhook"
            type="url"
            placeholder="https://hooks.slack.com/services/..."
            value={slackWebhookUrl}
            onChange={(e) => setSlackWebhookUrl(e.target.value)}
            className="font-mono text-sm rounded-xl border-border/60 bg-background h-10"
          />
          <p className="text-xs text-muted-foreground">
            Create an incoming webhook at{" "}
            <a
              href="https://api.slack.com/messaging/webhooks"
              target="_blank"
              rel="noopener noreferrer"
              className="text-primary underline underline-offset-2"

        <Button

          className="bg-primary hover:bg-primary/90 text-white rounded-xl h-10 px-6 font-semibold"
    ```
